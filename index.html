1. Pengertian

Algoritma sekuensial adalah jenis algoritma yang mengeksekusi instruksi secara berurutan, satu per satu, dari awal hingga akhir.

Konsep ini merupakan bagian fundamental dari dasar-dasar pemrograman terstruktur yang harus dikuasai setiap programmer.

2. Contoh Kasus

Menghitung jual beli susu sapi segar.

DEKLARASI

total : float

ALGORITMA
input nama
input tanggal
input jumlah

1. Pengertian

Algoritma perulangan for adalah struktur kontrol dalam pemrograman yang digunakan untuk mengeksekusi suatu blok kode secara berulang dengan jumlah iterasi yang sudah diketahui atau ditentukan sebelumnya.

  • Inisialisasi
  • Kondisi
  • Increment / Decrement

Menghitung total liter susu yang terjual selama 7 hari.

# Menghitung total penjualan susu selama 7 hari

total = 0

for i in range(1, 8):
    liter = float(input(f"Masukkan penjualan hari ke-{i} (liter): "))
    total = total + liter

print("\nTotal susu terjual selama 7 hari =", total, "liter")

1. Pengertian

Algoritma perulangan while adalah salah satu bentuk looping dalam pemrograman yang digunakan untuk menjalankan suatu blok kode berulang kali selama kondisi tertentu bernilai benar.

  • Kondisi dicek di awal.
  • Jumlah perulangan tidak pasti.
  • Bergantung pada kondisi tertentu.

2. Contoh Kasus

Menghitung total pembelian susu sampai pembeli selesai berbelanja.

3. Pseudocode

PROGRAM Menghitung Total Pembelian Susu

DEKLARASI
jumlah : float
total : float

total ← 0

input jumlah

WHILE jumlah ≠ 0 DO
    total ← total + jumlah
    input jumlah
END WHILE

output total

SELESAI

4. Program Python

# Menghitung total pembelian susu

total = 0

jumlah = float(input(
"Masukkan jumlah liter susu (0 untuk selesai): "
))

while jumlah != 0:
    total = total + jumlah

    jumlah = float(input(
    "Masukkan jumlah liter susu (0 untuk selesai): "
    ))

print("\nTotal susu yang dibeli =", total, "liter")

Output Program

Masukkan jumlah liter susu (0 untuk selesai): 2
Masukkan jumlah liter susu (0 untuk selesai): 1.5
Masukkan jumlah liter susu (0 untuk selesai): 3
Masukkan jumlah liter susu (0 untuk selesai): 0

Total susu yang dibeli = 6.5 liter

1. Pengertian

Algoritma perulangan do-while adalah bentuk pengulangan yang menjalankan blok kode minimal satu kali, lalu memeriksa kondisi di akhir perulangan untuk menentukan apakah proses akan diulang kembali.

Ciri Utama

  • Kondisi diperiksa di akhir (post-test loop).
  • Blok kode dijalankan minimal satu kali.
  • Cocok digunakan untuk validasi input.

2. Program Python

# Contoh simulasi do-while di Python

while True:
    try:
        angka = int(input(
        "Masukkan angka positif: "
        ))

    except ValueError:
        print("Input harus berupa angka!")
        continue

    if angka > 0:
        print(f"Anda memasukkan angka {angka}")
        break
    else:
        print("Angka harus positif, coba lagi.")

Output Program

Masukkan angka positif: 10
Anda memasukkan angka 10

Penjelasan Kode

  • while True membuat loop tanpa batas.
  • break menghentikan perulangan.
  • continue mengulangi perulangan.
  • try-except menangani kesalahan input.

1. Pengertian

Perulangan bersarang (nested loop) adalah struktur perulangan di dalam perulangan lain.

Satu loop akan berjalan, kemudian di dalamnya terdapat loop lain yang berjalan untuk setiap iterasi loop luar.

Kegunaan Nested Loop

  • Mengakses matriks atau data 2 dimensi.
  • Membuat pola bintang.
  • Kombinasi dan permutasi.

2. Contoh Kasus

Mencatat penjualan susu selama 3 minggu, dimana setiap minggu terdiri dari 7 hari.

  • Loop luar = minggu
  • Loop dalam = hari
